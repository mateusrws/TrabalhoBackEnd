BACK END - INFORMAÇÕES

ROTAS


CLIENTE

GET - /cliente || Retorna uma lista com todos os clientes

POST - /cliente || Cria uma conta de um novo cliente

PUT - /cliente/:cli_cod || Atualiza dados de um cliente já existente

DELETE - /cliente/:cli_cod || Deleta o perfil de um cliente já existente

PRODUTOS

GET - /produtos || Retorna uma lista com todos os produtos existentes

POST - /produtos || Cadastra um novo produto no sistema

PUT - /produto/:pro_cod || Atualiza dados de um produto já existente

DELETE - /produto/:pro_cod || Deleta um produto já existente

VENDAS

GET - /vendas/:cli_cod || Retorna uma lista de todas a vendas de um cliente específico

POST - /vendas || Cadastra uma venda

TABELAS

CLIENTES

CLI_COD - serial PRIMARY KEY

CLI_NOME - VARCHAR(20) NOT NULL

CLI_EMAIL - VARCHAR(100) NOT NULL

CLI_TEL - VARCHAR(20) NOT NULL

PRODUTOS

PRO_COD - serial PRIMARY KEY

PRO_NOME - VARCHAR NOT NULL

PRO_DESCRI - TEXT NOT NULL

PRO_FABRICANTE VARCHAR(255) NOT NULL

PRO_QTDA - INTEGER

PRO_PRECO - NUMERIC(10,2)

VENDA

CLI_COD - INTEGER NOT NULL

PRO_COD - INTEGER NOT NULL

VEN_COD - serial PRIMARY KEY

VEN_DATA - TIMESTAMP NOT NULL CURRENT_TIMESTAMP

VEN_VALOR - DECIMAL(10,2)
